About

This is a Linked Data Event Stream (LDES) feed for the EU Corporate Body vocabulary, published as SKOS concepts by the Publications Office of the European Union.

Usage

To consume this feed programmatically, use the following entry point:

https://ValyVanDenBroeck.github.io/ldes-training-project/index.trig

Format

The feed is published in TriG format (RDF 1.1), which extends Turtle to support named graphs.
